# Neural Voices Speech Dispatcher
* Разработанно Smit TV
## Сборка и Установка
* Чтобы собрать и установить проект вам необходимо воспользоваться скриптом setup.sh
* Чтобы добавить голос создайте директорию /usr/local/neuralvoices и в нутри нее положите каталоги с голосами в и файлом model.key для каждого
